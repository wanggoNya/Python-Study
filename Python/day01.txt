파이썬이란 ??
	- 1991년 네덜란드 수학자 '귀도 반 로섬'에 의해 개발된 프로그래밍 언어
	- '비단뱀'에서 유래한 로고
	- 구글의 많은 앱들과 인스타그램 등등의 실용적인 앱들도 파이썬으로 만들어졌다.

장점
	1) 인터프리터 언어
	Line by line 한 줄씩 실행되어 결과바로 확인할 수있음
	2) 플랫폼에 독립적이다
	어떠한 운영체제에서도 같은 파이썬 코드 실행 가능
	다양한 운영체제(윈도우, osx, 리눅스 등)에서 동일한 코드, 동일한 결과 볼 수 있음
	3) 객체 지향 언어이다
object oriented 클래스와 객체를 기반으로하는 언어로 정보은닉, 상속 등의 개념을 지원한다.
	4) 동적 타이핑 dynamic typing 언어이다.
프로그램 실행 중에 다른 작업 처리가능
	5) 개발 속도빠름
이미 완성된 많은 다양한 라이브러리, 데이터 분석 pandas, 수치연산 numpy, 시각화 matplotlib, Django, falsk, 웹개발, science kit learning 등등을 사용해 빠르게 프로그램 작성 가능

단점
	1) 하드웨어, 앱개발(아이폰, 안드로이드) 등에는 적용하기 어렵다.

주석
	• 프로그램 소스 코드에 설명을 작성하는 것
	• 프로그램 실행에 전혀 영향을 끼치지 않음
	• 유지 보수나 팀 작업을 할 때 모두가 이해하기 쉽고, 
의미 전달이 잘 되는 주석을 작성할 것
	• 파이썬에서 주석 : # 또는''' (삼중따옴표)
	
변수
	• 어떤 데이터를 사용하는 메모리 저장소
	• 변수 만들기 : 변수명 = 값
	
	- 규칙
	1) 영문, 한글, 숫자, 밑줄로 구성
	2) 특수문자사용X
	3) 대문자, 소문자 구분(가급적 소문자 사용)
	4) 숫자로 첫글자 불가
	5) 예약어(print, list, dict, if, for 등)
	• 의미있는 변수명 사용하자

기본 자료형 type
	• 정수 int 1, 10, 15
	• 실수 float 3.14, 2.7
	• 논리 bool True(1) False(0)
	• 문자열 str '홍길동' 따옴표가 붙으면 문자!!!! (큰 따옴표도 ok)

인덱싱 indexing
	• 문자열, 리스트, 튜플 모두 동일
	• 몇 번째인지 알려주는 고유한 번호
	• 파이썬은 0에서 시작한다.
	• H e  l  l  o
	• 0 1 2 3  4
	• -5-5-3-2-1
	
슬라이싱 
	• 문자열, 리스트, 튜플 모두 동일
	• 한 문자 이상으로 구성된 단어, 문장 등을 추출할 때 사용
	• 0번째부터 2번째 미만까지 [start:stop:step] step은 생략가능
